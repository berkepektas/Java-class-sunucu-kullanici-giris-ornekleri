# JavaClassSunucuKullaniciGiris

Bu proje, Java kullanılarak geliştirilmiş bir **kullanıcı giriş sistemi örneğidir**. 
Sunucu ile veri iletişimi ve kullanıcı doğrulama işlemleri üzerinde yoğunlaşmıştır. 
Proje, temel Java sınıf ve fonksiyon yapılarının kullanımı ile sunucu-tabanlı uygulama geliştirme becerilerini artırmayı hedefler.

## Özellikler
- Kullanıcı kayıt ve giriş doğrulama
- Sunucuya veri gönderme ve alma
- Basit ve anlaşılır Java sınıf yapısı

## Kurulum
1. Projeyi klonlayın: `git clone <repo_link>`
2. IDE veya terminal üzerinden Java dosyalarını çalıştırın

## Kullanım
- Kullanıcı adı ve şifre ile giriş yapabilirsiniz
- Sunucuya mesaj gönderip alabilirsiniz
